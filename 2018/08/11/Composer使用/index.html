Composer使用

2018.08.11 Xj Laravel  热度 ℃

通过Composer安装laravel框架

  • 1.通过Composer create-project 命令安装laravel

composer create-project laravel/laravel project_name –prefer-dist

  • 2.通过laravel安装器安装laravel(需要手动添加环境变量)

composer global require “laravel/installer”
laravel new blog
vim ~/.bash_profile //(环境变量添加)在终端执行该命令
//在该文件中添加: export PATH=~/.composer/vendor/bin

Artisan控制台

  • Artisan是laravel中自带的命令行工具,由强大的Symfony Console组件驱动,提供了一些对应用开发有帮助的命令
  • Artisan使用

//查看所有可用Artisan命令(list)
php artisan
php artisan list

//查看命令的帮助信息(help)
php artisan help migrate

//创建控制器
php artisan make:controller UserController

//创建模型
php artisan make:model Info

//创建中间件
php artisan make:middleware Activity

Laravel中的用户认证(Auth)

  • 生成Auth所需文件

    php artisan make:auth

可在路由中看到: 访问home即可看到用户登录界面

工作目录下执行

php artisan migrate

即可创建相应的数据表,再次进入用户登录界面即可进行注册登录操作

  • 数据迁移
  1. 新建一个students表的迁移文件

php artisan make:migration create_students_table –create=students

–table 和 –create 参数可以用来指定数据表名称,以及迁移文件是否要建立新的数据表。

  1. 生成模型的同事生成迁移文件

php artisan make:model Student -m

在 database/migrations 目录下找到相应的迁移文件并修改:

在工作目录下执行

php artisan migrate

即可在本地数据库创建students表

  • 数据填充
  1. 创建一个填充文件,并完善填充文件 文件目录: database/seeds

php artisan make:seeder StudentTableSeeder

创建成功后在run()方法中插入数据

  1. 执行单个填充文件

php artisan db:seed –class=StudentTableSeeder

  1. 批量执行填充文件

php artisan db:seed
//需要在 database/seeds/DatabaseSeeder.php 中引入该填充文件:
$this->call(StudentTableSeeder::class);
