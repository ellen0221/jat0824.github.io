Laravel-admin学习

2020.02.22 Xj Laravel  热度 ℃

文档

https://laravel-admin.org/docs/zh

常用命令

  • 安装

    composer require encore/laravel-admin
    php artisan vendor:publish –provider=”Encore\Admin\AdminServiceProvider”
    php artisan admin:install

  • 添加控制器

    php artisan admin:make UserController –model=App\User //Linux系统需要转义反斜杠

  • 添加行操作

    php artisan admin:action Paper\Analysis –grid-row –name=”考卷分析”
