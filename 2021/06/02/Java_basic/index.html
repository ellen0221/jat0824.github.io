Java基础知识

2021.06.02 Xj Java  热度 ℃

JAVA特性：

  1. 纯面向对象：万物皆对象，三大特性——封装、继承、多态

  2. 跨平台的语言（提供了在不同平台下运行的解释环境——JVM）

  3. 健壮的语言，许多底层是通过C/C++实现，同时在C/C++语言的基础上进行了优化

  4. 较高安全性——自动回收垃圾（监控程序运行中开辟的内存，检查并对可释放的内存即垃圾进行释放）、强制类型检查（减少错误发生的可能）、取消指针

JAVA跨平台原理

JAVA写的程序可以在不同的平台（Windows、Mac OS、Linux等）运行

JDK与JRE

JDK（Java Development Kits），Java开发工具集，包括：

  1. JRE（Java Runtime Enviornment）Java运行时环境
  2. JAVA API(应用程序编程接口)、工具程序——Java编译器(javac.exe)、Java运行时解释器(java.exe)、Java文档化工具(javadoc.exe)及其他工具资源

JRE包括：

  1. JVM——执行代码（运行时解释器完成）
  2. 系统类库

JRE主要功能：

  1. 加载代码：类加载器——用于把.class文件加载到JVM中
  2. 校验代码：字节码校验器
  3. 执行代码：运行时解释器

JRE针对使用者——运行java程序，JDK针对开发者——编译及运行

语法

  1. 严格区分大小写
  2. 代码分为结构定义语句(花括号外的语句)、功能执行语句(花括号内的语句，以分号结尾)
