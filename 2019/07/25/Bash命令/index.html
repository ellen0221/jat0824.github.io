Bash命令

2019.07.25 Xj  热度 ℃
  • ctrl+w: 删除你键入的最后一个单词
  • ctrl+u: 删除行内光标所在位置之前的内容
  • alt+b和alt+f: 以单词为单位移动光标
  • ctrl+a: 将光标移至行首
  • ctrl+e: 将光标移至行尾
  • ctrl+k: 删除光标至行尾的所有内容
  • ctrl+l: 清屏
  • alt+.: 循环地移向前一个命令
  • ctrl+r: 搜索命令行历史记录:重复按下 ctrl+r 会向后查找匹配项，按下 Enter 键会执行当前匹配的命令，而按下右方向键会将匹配项放入当前行中，不会直接执行，以便做出修改
  • cd -: 回到前一个工作路径
  • man ascii: 查看具有十六进制和十进制值的ASCII表
  • w: 查看系统已经运行多长时间
  • top: 获取 CPU 和硬盘的使用状态
  • netstat: 查看网络连接的细节
  • sw_vers: 获取 OS X 的版本信息
