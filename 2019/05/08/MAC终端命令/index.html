    将文件内容传输到剪切板

    cat a.txt | pbcopy // 将a.txt文件类容写入剪切板，相当于 cmd + C
    pbpaste // 将刚刚存入剪切板的a.txt内容复制到终端，相当于 cmd + v

    从远程Linux服务器上传下载文件–scp命令
    • 下载文件夹

      scp -r 远程登录服务器用户名@远程服务器ip地址:/下载文件夹的目录 本地目录

    • 下载文件

      scp 远程登录服务器用户名@远程服务器ip地址:/下载文件的目录 本地目录

    • 上传文件夹

      cd 上传文件的目录
      scp -r 本地目录 远程登录服务器用户名@远程服务器ip地址:/文件上传的目录地址

    终端命令

    • 复制整个文件夹

      cp -R 路径1 路径2 # -R是对目录进行递归操作

    • 查看php扩展

      php -i |less // 查看配置文件在哪里，编译参数
      php -m |less // 查看php加载的模块
