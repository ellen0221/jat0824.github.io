二、curl请求api并获得返回数据

终端curl命令:

  • HTTP或FTP认证

    curl -u user:pwd http://xxx.com // 指定用户名和密码
    curl -u user http://xxx.com // 指定用户名，在后续操作中输入密码

  • 发送GET请求

    curl http://xxx.com

  • 发送POST请求

    curl -d “key1=value1&key2=value2” http://xxx.com

  • 添加HTTP header

    curl -H ‘Authorization: xxx’ -H ‘Content-Type: application/json’ -v http://xxx.com
    或 curl –header ‘Authorization: xxx’ “http://xxx.com“

三、crontab学习

crontab命令

crontab [-u user] file

或

  • -e : 执行文字编辑器来设定时程表，内定的文字编辑器是 VI，如果你想用别的文字编辑器，则请先设定 VISUAL 环境变数来指定使用那个文字编辑器(比如说 setenv VISUAL joe)
  • -r : 删除目前的时程表
  • -l : 列出目前的时程表

crontab定时任务

用crontab -e进入当前用户的工作表编辑，是常见的vim界面。每行是一条命令。
crontab的命令构成: 时间+动作
时间有 分、时、日、月、周 五种，操作符有

  • *：取值范围内的所有数字
  • /：每过多少个数字
  • -：从X到Z
  • ,：散列数字

例1：每周六、日的晚上11点到早上7点的第3和第15分钟执行

3,15 23-7 6,0 myCommand

例2：每隔两天的上午8点到11点的第3和第15分钟执行

3,15 8-11 /2 * myCommand

vim使用技巧：

编辑后：esc + :w 不要使用wq 可保存当前编辑 若出现错误可以按u撤回修改
