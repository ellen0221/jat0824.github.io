git命令

2019.01.26 Xj  热度 ℃

常用

  • 初始化仓库(进入项目根目录)

    git init

  • 将所有文件提交到索引仓库

    git add .

  • 将索引仓库中的文件添加到git仓库

    git commit -m “message”

  • 将master分支推送到远程仓库

    git push origin master

  • 将本地仓库追踪到远程仓库

    1. 首先在自己的github上新建一个项目得到项目地址(例如:https://github.com/jat0824/process_schedule.git)
    2. 初始化本地仓库后，将本地master分支追踪到远程分支，运行以下命令:

      git remote add origin https://github.com/jat0824/process_schedule.git

    3. 将本地文件提交到仓库并推送到远程仓库

      git add .
      git commit -m “message”
      git push origin master

  • 新建分支并切换到新分支

    git branch -b newbranch
